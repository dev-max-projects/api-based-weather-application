# 🌦️ Weather Application

A simple **Weather Application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using an **API**.  
Users can enter a city name to check the current weather conditions.

## Features
- Search weather by city name
- Display temperature
- Show weather condition
- Simple and clean interface
- Real-time data using API

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Enter a city name to get weather information.
